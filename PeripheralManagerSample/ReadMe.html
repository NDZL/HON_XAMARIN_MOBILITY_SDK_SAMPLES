Sample Overview

 

This PeripheralManagerSample application is a Xamarin.Forms application that was built with Microsoft Visual Studio 2017. It demonstrates dock type and serial port information by using the API provided by Honeywell Mobility OS SDK for Xamarin.

 

This sample application contains a Peripheral Manager Sample single screen:

1.     Click the DOCK TYPE button to display the type of the dock connected to the device.

2.     Click the SERIAL INFO button to display the serial port information of the device.

 

Runtime Requirements and Dependencies

 

  • Please see the OS SDK release notes for platform requirements.
  • This sample application depends on the Droid.OS.dll which is included in the Honeywell.AIDC.Droid.OS.x.x.x.x.nupkg where x.x.x.x is the package version. Please refer to the Installing SDK NuGet Package section of the Honeywell Mobility OS SDK for Xamarin API Guide for how to install the SDK NuGet package.

 

Building the Sample

 

The PeripheralManagerSample projects can be built in the standard manner:

1.     Load the PeripheralManagerSample.sln with Visual Studio .NET 2017.

2.     Make sure the Solution Platform is set to: Any CPU

3.     Under the Build tab of the toolbar, click Rebuild Solution.

4.     If Visual Studio does not automatically restore the NuGet packages referenced by the sample application during a build, right click on the PeripheralManagerSample.Android project from Solution Explorer and select the Manage Nuget Packages to install the Honeywell.AIDC.Droid.OS nugget package, the Honeywell.AIDC.Droid.OS should be added to the References under PeripheralManagerSample.Android project if it is installed successfully.

 

Running the Sample

For Android computer:

  • Make sure the USB debugging is enabled from Setting > Developer options on the Android computer.
  • Connect the Android computer to your PC.
  • In the Visual Studio toolbar, press the Start button (the triangular button that resembles a Play button) to deploy and start the application on the Android computer.
